**Website institucional e portal administrativo desenvolvido para Vedavel Vedações Industriais.**

O projeto foi desenvolvido para modernizar a presença digital da empresa, permitindo a apresentação institucional, catálogo de produtos e gerenciamento interno através de um portal administrativo.

---

## Objetivo do Projeto

O **Vedavel — Vedações Industriais** foi criado com o objetivo de resolver necessidades operacionais e digitais da empresa:

- Modernização da presença digital
- Centralização das informações institucionais
- Facilidade no gerenciamento de produtos
- Organização dos catálogos técnicos
- Melhor comunicação com clientes

O sistema centraliza as informações da empresa em um único ambiente web moderno e responsivo.

## Arquitetura

O projeto foi desenvolvido utilizando estrutura modular em PHP:

- **Frontend** → Interface institucional do website
- **Backend** → Regras administrativas
- **Portal Admin** → Gerenciamento de produtos e catálogos
- **Banco de Dados** → Armazenamento das informações do sistema
